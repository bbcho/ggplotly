# Gallery

A visual gallery of ggplotly examples organized by category. Each example includes complete, runnable code.

## Categories

- [Basic Charts](basic.md) - Scatter, line, bar, histograms
- [Statistical Visualizations](statistical.md) - Smoothing, density, distributions
- [Time Series](timeseries.md) - Date axes, range plots, financial charts
- [Geographic Maps](maps.md) - Choropleths, projections, point maps
- [3D Visualizations](3d.md) - Scatter, surfaces, wireframes
- [Network Graphs](networks.md) - Edge bundling, sea routes
- [Multi-Panel Plots](facets.md) - Faceting and small multiples
- [Theming & Styling](theming.md) - Custom themes, publication-ready charts
