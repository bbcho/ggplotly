# Gallery

A visual gallery of ggplotly examples organized by category. Each example includes complete, runnable code with rendered chart outputs.

## Categories

- [Basic Charts](basic.ipynb) - Scatter, line, bar, histograms
- [Statistical Visualizations](statistical.ipynb) - Smoothing, density, distributions
- [Time Series](timeseries.ipynb) - Date axes, range plots, financial charts
- [Geographic Maps](maps.ipynb) - Choropleths, projections, point maps
- [3D Visualizations](3d.ipynb) - Scatter, surfaces, wireframes
- [Network Graphs](networks.ipynb) - Edge bundling, sea routes
- [Multi-Panel Plots](facets.ipynb) - Faceting and small multiples
- [Theming & Styling](theming.ipynb) - Custom themes, publication-ready charts
